# acyr — brand docs

Identity, voice, and asset specs. Most files mirror `src/brand/data/*.js` — JS is canonical, these summarise for human scanning.

## Files

- [[02-brand/02-career-and-press|career-and-press]] — bio, education, shows, press citations, awards. Mirrors `business-data.js`.
- [[02-brand/03-branded-assets|branded-assets]] — physical print specs (stationery, labels, packaging, soft goods). Mirrors `branded-assets.js`.
- [[02-brand/04-writing-guidelines|writing-guidelines]] — voice and tone manifesto. Quiet, considered, anti-trend.
- [[02-brand/05-writing-examples|writing-examples]] — ten worked prose pieces showing the voice in use.

- [[02-brand/07-newsletter-template|newsletter-template]] — monthly newsletter structure with placeholder body + worked June example. Voice-anchored to writing-guidelines.
